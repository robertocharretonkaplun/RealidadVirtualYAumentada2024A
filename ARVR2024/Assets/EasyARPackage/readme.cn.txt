EasyAR Sense Unity Plugin
使用 UPM (https://docs.unity3d.com/Manual/upm-ui.html) 导入package。
    com.easyar.sense：EasyAR Sense Unity Plugin主体。
    com.easyar.sense.ext.arfoundation：EasyAR Sense Unity Plugin ARFoundation 扩展，需要使用AR Foundation时导入，需要先导入com.easyar.sense。
    com.easyar.sense.ext.nreal：EasyAR Sense Unity Plugin Nreal 扩展，需要使用Nreal时导入，需要先导入com.easyar.sense和NRSDK。
请阅读EasyAR网站上的文档来了解应该如何使用：https://www.easyar.cn/view/support.html
