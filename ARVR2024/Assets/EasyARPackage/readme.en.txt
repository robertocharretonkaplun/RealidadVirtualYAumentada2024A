EasyAR Sense Unity Plugin
Use UPM (https://docs.unity3d.com/Manual/upm-ui.html) to import packages.
    com.easyar.sense: EasyAR Sense Unity Plugin main package.
    com.easyar.sense.ext.arfoundation: EasyAR Sense Unity Plugin ARFoundation Extension, import when AR Foundation is to be used. com.easyar.sense must be imported first.
    com.easyar.sense.ext.nreal: EasyAR Sense Unity Plugin Nreal Extension, import when Nreal is to be used. com.easyar.sense and NRSDK must be imported first.
Please read the documents on EasyAR website for how to use, https://www.easyar.com/view/support.html
